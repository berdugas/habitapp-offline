
Habit Creation
  -  Habit should be grounded on a goal of becoming something, someone or a transformation
  -  To become someone or transform to be better you need to change a habit or a group of habits that relate to each other
  -  This is where there is a core habit and the supporting 2 habits
  -  Idea / Transformations > Core Habit > Supporting Habits
  -  When helping the user build habits, we first introduce a core habit, and when he gets a certain consistency, start to introduce supporting habits. 
  -  The user can decline to have supporting habits.

Adding Habits
